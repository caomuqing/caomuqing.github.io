üìã Quick Image Creation Guide

Option 1: Online Tools (Recommended)

  • Canva.com - Free templates, easy to use
  • Figma.com - Professional design tool
  • GIMP - Free desktop image editor

Option 2: Use the Python Script

  • Install Pillow: pip install Pillow
  • Run: python generate_publication_images.py
  • Customize generated images as needed

Option 3: Manual Creation

  • Use the color palettes above
  • Follow the theme guidelines
  • Create 500x300 pixel images
  • Save as PNG format
